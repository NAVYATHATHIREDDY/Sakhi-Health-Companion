# 🌸 Sakhi — Your Personal Health Companion

A Python-based women's health tracker built with Tkinter GUI.

## Features
- 📅 Period date tracking and next period prediction
- 😊 Daily mood logger
- 🤒 Symptoms tracker
- 💾 Data saving to local file
- 🌸 Beautiful pink and white GUI

## Tech Stack
- Python 3
- Tkinter (GUI)
- Datetime module
- File handling

## How to Run
```bash
python3 sakhi_gui.py
```
